## Zigbee2MQTT

Zigbee2MQTT is not bundled in this repository.

The NewGUI WebUI workflow was inspired by the way Zigbee2MQTT separates the
browser frontend from backend state and live updates. Based on the current local
comparison against `zigbee2mqtt-master`, this repository does not include copied
Zigbee2MQTT source code or assets.

If copied or modified Zigbee2MQTT code/assets are ever introduced, their GPL-3.0
